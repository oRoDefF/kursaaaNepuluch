ПАМЯТКА ПОЛЬЗОВАТЕЛЯ: ИНТЕРНЕТ-МАГАЗИН ЭЛЕКТРОНИКИ
1. ДОБАВЛЕНИЕ ТОВАРОВ
1.1. Через административную панель:
   - Войдите в админ-панель: http://localhost:8000/admin/
   - Используйте логин и пароль суперпользователя
   - Перейдите в раздел "Товары" (Products)
   - Нажмите "Добавить товар" (Add Product)
   - Заполните поля:
     * Название (Name) - обязательное поле
     * Категория (Category) - выберите из списка
     * Описание (Description) - необязательное
     * Цена (Price) - обязательное поле
     * Изображение (Image) - загрузите файл
     * Доступен (Available) - отметьте галочкой
   - Нажмите "Сохранить"
1.2. Автоматическое создание slug:
   - Slug создается автоматически из названия товара
   - Можно изменить вручную в админ-панели
2. УДАЛЕНИЕ ТОВАРОВ
2.1. Через административную панель:
   - Перейдите в раздел "Товары"
   - Найдите нужный товар
   - Нажмите на название товара для редактирования
   - Внизу страницы нажмите "Удалить" (Delete)
   - Подтвердите удаление
2.2. Массовое удаление:
   - В списке товаров отметьте галочками нужные товары
   - В выпадающем списке "Действие" выберите "Удалить выбранные товары"
   - Нажмите "Выполнить"
3. РЕДАКТИРОВАНИЕ ТОВАРОВ
3.1. Изменение информации:
   - Найдите товар в списке
   - Нажмите на название для редактирования
   - Измените нужные поля
   - Нажмите "Сохранить"
3.2. Быстрое редактирование в списке:
   - В списке товаров можно быстро изменить:
     * Цену (Price)
     * Статус доступности (Available)
   - Просто кликните на поле и измените значение
4. УПРАВЛЕНИЕ КАТЕГОРИЯМИ
4.1. Добавление категории:
   - Перейдите в раздел "Категории" (Categories)
   - Нажмите "Добавить категорию"
   - Заполните:
     * Название (Name)
     * Slug (создается автоматически)
   - Сохраните
4.2. Редактирование категории:
   - Найдите категорию в списке
   - Нажмите на название для редактирования
   - Измените нужные поля
   - Сохраните
5. УПРАВЛЕНИЕ ЗАКАЗАМИ
5.1. Просмотр заказов:
   - Перейдите в раздел "Заказы" (Orders)
   - Просматривайте список всех заказов
   - Используйте фильтры по статусу, дате, оплате
5.2. Изменение статуса заказа:
   - Откройте заказ для редактирования
   - В поле "Статус" выберите новый статус:
     * Ожидает обработки (pending)
     * В обработке (processing)
     * Отправлен (shipped)
     * Доставлен (delivered)
     * Отменен (cancelled)
   - Сохраните изменения
5.3. Отметка оплаты:
   - В заказе отметьте галочку "Оплачен" (Paid)
   - Сохраните изменения
6. УПРАВЛЕНИЕ КОРЗИНАМИ
6.1. Просмотр корзин пользователей:
   - Перейдите в раздел "Корзины" (Carts)
   - Просматривайте корзины всех пользователей
   - Видите общую стоимость и количество товаров
6.2. Просмотр элементов корзины:
   - Перейдите в раздел "Элементы корзины" (Cart Items)
   - Видите все товары в корзинах пользователей
7. ПОЛЕЗНЫЕ КОМАНДЫ
7.1. Заполнение тестовыми данными:
   python manage.py populate_db
7.2. Создание суперпользователя:
   python manage.py createsuperuser
7.3. Применение миграций:
   python manage.py migrate
7.4. Создание миграций:
   python manage.py makemigrations
7.5. Запуск сервера:
   python manage.py runserver
8. СОВЕТЫ ПО ИСПОЛЬЗОВАНИЮ
8.1. Изображения товаров:
   - Поддерживаются форматы: JPG, PNG, GIF
   - Рекомендуемый размер: 800x600 пикселей
   - Файлы сохраняются в папке media/products/
8.2. Цены:
   - Указывайте цены в рублях
   - Используйте точку для разделения копеек
   - Пример: 129999.00
8.3. Описания товаров:
   - Используйте HTML-теги для форматирования
   - Рекомендуемая длина: 200-500 символов
8.4. Категории:
   - Создавайте логичные категории
   - Используйте понятные названия
   - Slug создается автоматически из названия
9. БЕЗОПАСНОСТЬ
9.1. Пароли:
   - Используйте сложные пароли
   - Регулярно меняйте пароли
   - Не передавайте пароли третьим лицам
9.2. Резервное копирование:
   - Регулярно делайте резервные копии базы данных
   - Сохраняйте важные файлы
10. УСТРАНЕНИЕ ПРОБЛЕМ
10.1. Товар не отображается:
   - Проверьте, отмечена ли галочка "Доступен"
   - Убедитесь, что товар привязан к категории
10.2. Изображение не загружается:
   - Проверьте формат файла
   - Убедитесь, что файл не поврежден
   - Проверьте права доступа к папке media
10.3. Ошибки в форме заказа:
   - Проверьте корректность введенных данных
   - Убедитесь, что все обязательные поля заполнены
